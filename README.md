
# Symantix v6 — Insight Suite (Creative Cyber Dashboard)

Neon-glass cyber UI with a cinematic dashboard plus Insight pages: Timeline, Compliance, Reports (CSV export), Network Graph, Settings.

## Run locally
```bash
python3 -m http.server 8080
# open http://localhost:8080/index.html
```

## Highlights
- 3D-glass dashboard widgets (donut, line chart, KPIs, live feed)
- IAM, Vault, Network, Learning, Copilot core modules
- Insight Suite: Timeline, Compliance, Reports, Graph, Settings
- Theme toggle (Dark Neon / Pure Black / Dim Navy)
- PWA offline cache + installable
